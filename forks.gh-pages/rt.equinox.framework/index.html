Welcome

This is the
Eclipse Equinox Project

Eclipse Equinox is a GitHub fork
of the official upstream repository.
This is the repo where the work from EquinoxOs2
will be integrated with the intention
to present OS/2 as a new platform
for Eclipse.

Eclipse Equinox has its focus on upstream integration

Developer Notes

Because this project is a fork,
the master branch follows upstream:master and
the integration for OS/2 will take place
at the master-os2 branch.

Commits to this repo
will start after EquinoxOs2 has reached
a more mature level and things start falling
into place. Then, regular integration
commits will be made.

For now, this repository is idle
